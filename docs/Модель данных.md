## Position (Позиция)

- **id** : SERIAL (PRIMARY KEY) - идентификатор позиции
- **layout** : JSONB (NOT NULL) - данные о позиции, расположение и тип элементов
- **power** : DOUBLE PRECISION - сила удара
- **created_at** : TIMESTAMP (DEFAULT now()) - дата и время создания
- **updated_at** : TIMESTAMP (DEFAULT now()) - дата и время обновления
