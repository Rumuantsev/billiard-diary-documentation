# Глоссарий

| Термин         | Определение                                                                                                            |
| -------------- | ---------------------------------------------------------------------------------------------------------------------- |
| **Биток**      | Шар, по которому производится удар                                                                                     |
| **Винт**       | Точка попадания по битку                                                                                               |
| **Позиция**    | Схема, отображающая расположение шаров на столе, также включающая в себя винт и силу удара. Является частью упражнения |
| **Упражнение** | Задание от тренера к ученику, включающее название, а также опционально: описание, позицию, фото, видео                 |
| **Тренировка** | Набор упражнений с количеством их выполнения                                                                           |
